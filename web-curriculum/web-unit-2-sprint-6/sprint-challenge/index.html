Sprint Challenge - Web Unit 2 Sprint 6

Sprint Challenge

Sprint Challenges test your skills on the topics presented during the Sprint. The challenges can be complex but rewarding, as they allow you to apply your newly acquired skills to unique scenarios. Completing all the challenges successfully is one of the milestones to move to the next Sprint.

Getting Started

Follow these steps to start the challenge and submit your code for grading:

  1. Use the Sprint Challenge link to obtain the starter code.
  2. Fork and Clone the project, open it in VS Code, and follow the README instructions to get started.

Sprint Challenge Submission

Submit your code for grading by pushing your commits to GitHub and submitting the URL to the Sprint Challenge tab in the BloomTech Portal.

If you did not receive a perfect score, don't worry! To re-submit, you only need to push additional commits using git push origin main until you achieve a 100%. Make sure to carefully review your code and the feedback provided by CodeGrade before resubmitting.
