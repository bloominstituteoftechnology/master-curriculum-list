What is a Code-Along?

Code-Alongs are live experiences designed to help you prepare for your upcoming Sprint Challenge. They are designed to be highly interactive, relevant to the content you are learning, and efficient in terms of time investment. Code-Alongs are not meant to be comprehensive or cover every possible topic in your Sprint. They are meant to provide you with a hands-on experience that will elevate your learning.

Common Data Structures Overview

During the Code-Along, we'll be working with several core data structures. Here's a brief overview:

Arrays

Arrays store ordered collections of values and are fundamental to many algorithms.

# Creating lists numbers = [1, 2, 3, 4, 5] mixed = [1, "two", {"three": 3}, [4]] # Common operations numbers.append(6) # Add to end - O(1) numbers.pop() # Remove from end - O(1) numbers.insert(0, 0) # Add to beginning - O(n) numbers.pop(0) # Remove from beginning - O(n) numbers.pop(2) # Remove at index - O(n) # Iteration for i in range(len(numbers)): print(numbers[i]) # Functional methods doubled = [num * 2 for num in numbers] evens = [num for num in numbers if num % 2 == 0] sum_total = sum(numbers)

Objects

Objects are collections of key-value pairs, providing fast access to values through their keys.

# Creating dictionaries person = { "name": "Alice", "age": 28, "isStudent": True } # Adding properties person["location"] = "New York" # Accessing properties print(person["name"]) # Bracket notation print(person.get("age")) # Safe access # Checking if property exists print("name" in person) # True # Iterating through properties for key in person: print(f"{key}: {person[key]}") # Getting keys and values keys = list(person.keys()) # ["name", "age", "isStudent", "location"] values = list(person.values()) # ["Alice", 28, True, "New York"]

Sets

Sets store unique values and are useful for removing duplicates and checking membership.

# Creating a set unique_numbers = set([1, 2, 3, 2, 1]) # {1, 2, 3} # Adding values unique_numbers.add(4) # {1, 2, 3, 4} unique_numbers.add(2) # No change, already exists # Checking for a value print(3 in unique_numbers) # True print(5 in unique_numbers) # False # Removing values unique_numbers.discard(1) # {2, 3, 4} # Size of set print(len(unique_numbers)) # 3 # Iterating through values for num in unique_numbers: print(num)

Maps

Maps are key-value collections where keys can be of any type, unlike objects where keys are strings.

# Creating a dictionary (map) user_roles = {} # Adding entries user_roles["alice"] = "admin" user_roles["bob"] = "editor" user_roles["charlie"] = "viewer" # Getting values print(user_roles.get("alice")) # "admin" print(user_roles.get("dana")) # None # Checking if key exists print("bob" in user_roles) # True # Size of map print(len(user_roles)) # 3 # Deleting entries user_roles.pop("charlie", None) # Iterating through entries for user, role in user_roles.items(): print(f"{user} is a {role}")
