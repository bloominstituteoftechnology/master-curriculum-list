Module 2: Events in the Browser

In this module, you'll learn about event handling in the browser. You'll understand different types of events, how to add event listeners, and how events propagate through the DOM.

Understanding Event Types

Learn about different types of events that can occur in the browser and how to handle them.

Handling Events with HTML Attributes

Learn how to handle events using HTML attributes and inline event handlers.

Handling Events with addEventListener

Master the use of addEventListener for handling events in JavaScript.

Understanding Event Propagation

Learn about how events propagate through the DOM tree and the different phases of event propagation.

Stopping Event Propagation

Learn how to control event propagation using stopPropagation and preventDefault.

Event Delegation

Learn about event delegation and how to use it to handle events on dynamically created elements.
