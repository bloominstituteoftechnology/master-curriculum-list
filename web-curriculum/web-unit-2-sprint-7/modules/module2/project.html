Module 2 Project - Form Management

Project Overview

In this project, you will build a React application that demonstrates your understanding of form management. You will create a complex form with validation, error handling, and proper state management.

Project Requirements

Form Requirements

  • Create a multi-step form with at least 3 steps
  • Implement form validation for all fields
  • Display validation errors in real-time
  • Handle form submission with proper error handling
  • Implement form state persistence between steps

Component Requirements

  • Create reusable form components
  • Implement controlled components for all form inputs
  • Create custom form validation hooks
  • Build a progress indicator for multi-step form

Functionality Requirements

  • Handle different input types (text, email, password, select, etc.)
  • Implement form data persistence using localStorage
  • Add form submission success/error feedback
  • Include form reset functionality

Project Setup

# Create a new React project
npx create-react-app form-management-project

# Install dependencies
cd form-management-project
npm install formik yup

# Start the development server
npm start

Example Implementation

// FormStep.js
import { useFormik } from 'formik';
import * as Yup from 'yup';

const validationSchema = Yup.object({
  firstName: Yup.string()
    .required('First name is required')
    .min(2, 'First name must be at least 2 characters'),
  lastName: Yup.string()
    .required('Last name is required')
    .min(2, 'Last name must be at least 2 characters'),
  email: Yup.string()
    .email('Invalid email address')
    .required('Email is required'),
});

function FormStep({ initialValues, onSubmit, children }) {
  const formik = useFormik({
    initialValues,
    validationSchema,
    onSubmit,
  });

  return (
    

      {children(formik)}
    

  );
}

export default FormStep;
