Sprint Challenge - Web Unit 1 Sprint 1

Overview

This sprint challenge will test your understanding of JavaScript fundamentals, HTML structure, and CSS styling. You'll build a responsive web application that demonstrates your knowledge of these core concepts.

Project Setup

Follow the video guide to set up your project and understand the requirements.

Challenge Requirements

  • Create a responsive web application using HTML, CSS, and JavaScript
  • Implement proper HTML structure with semantic elements
  • Style the application using CSS with a mobile-first approach
  • Add JavaScript functionality for user interaction
  • Ensure code follows best practices and is well-documented

Submission Guidelines

  • Submit your code through the provided GitHub repository
  • Include a README.md file with setup instructions
  • Ensure all files are properly organized and named
  • Test your application across different devices and browsers

Grading Rubric

Technical Requirements (40%)

  • HTML structure and semantics (10%)
  • CSS styling and responsiveness (15%)
  • JavaScript functionality (15%)

Code Quality (30%)

  • Code organization and structure (10%)
  • Naming conventions and readability (10%)
  • Comments and documentation (10%)

Project Submission (30%)

  • README.md completeness (10%)
  • GitHub repository organization (10%)
  • Overall project presentation (10%)
