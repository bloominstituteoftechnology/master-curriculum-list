Module 1: Database Basics

Learn the fundamentals of relational databases, SQL basics, and how to work with data using knex.js.

Learning Objectives

  • Explain what a relational database is and its core components
  • Understand SQL basics and its advantages
  • Learn to query, insert, and modify data in SQL
  • Write database queries using knex.js

Module Content

1. Introduction to Relational Databases

Learn about the fundamental concepts of relational databases, including:

  • What is a database?
  • Tables, rows, and columns
  • Primary and foreign keys
  • Relationships between tables

2. SQL Basics

Master the basics of SQL, including:

  • SELECT statements
  • WHERE clauses
  • JOIN operations
  • INSERT, UPDATE, and DELETE operations

3. Working with Knex.js

Learn how to use Knex.js for database operations:

  • Setting up Knex.js
  • Writing queries with Knex
  • Migrations and seeds
  • Best practices
