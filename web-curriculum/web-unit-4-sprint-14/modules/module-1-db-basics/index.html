Module 1: Database Basics

Module Overview

Learn the fundamentals of relational databases, SQL basics, and how to work with data using knex.js.

Learning Objectives

  • Explain what a relational database is and its core components
  • Understand SQL basics and its advantages
  • Learn to query, insert, and modify data in SQL
  • Write database queries using knex.js

Module Content

1. Introduction to Relational Databases

Learn about the fundamental concepts of relational databases, including:

  • What is a database?
  • Tables, rows, and columns
  • Primary and foreign keys
  • Relationships between tables

2. SQL Basics

Introduction to SQL language for managing database data:

  • Basic SQL commands
  • Creating and dropping databases and tables
  • Adding and modifying data
  • Simple queries and filters

3. Introduction to Knex.js

Learn how to use the Knex.js query builder for Node.js:

  • Setting up Knex.js in a project
  • Connecting to a database
  • Query building basics
  • Running SQL queries through Knex

4. Performing CRUD Operations

Learn how to perform basic database operations with Knex.js:

  • Creating data (INSERT)
  • Reading data (SELECT)
  • Updating data (UPDATE)
  • Deleting data (DELETE)
